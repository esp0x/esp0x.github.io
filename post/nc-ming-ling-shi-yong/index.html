nc命令使用

2021-02-03

端口扫描

# nc -vz -w 5 127.0.0.1 1-1024   // 扫描本地1-1024端口范围

监听端口

# nc -l 8000     // 监听TCP端口
# nc -ul 9999    // 监听UDP端口
# nc -vuz 127.0.0.1 9999   // 测试本地UDP端口

传输文件

1.上传
# nc -l 9999 > filename
# nc 127.0.0.1 9999 < source

2.下载
# nc -l 9999 < filename
# nc 127.0.0.1 9999 > target
